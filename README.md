# 基于 Krpano 的企业360全景线上展厅

## 1. 项目简介

本项目是一个基于 Web 技术的企业级 360 度全景线上展厅系统。主要功能包括：
- **前台展示端**：提供沉浸式的 360 度全景漫游体验、3D 模型（支持交互与拆解）在线预览、实时在线客服沟通以及用户留言功能。
- **后台管理端**：提供全景访客数据统计与图表分析、实时客服消息回复、留言审核管理、3D 模型参数配置及系统账号权限管理等功能。

项目采用前后端分离架构，前端使用 Vue3 + Vite + Element Plus + Three.js 构建，后端使用 Node.js (Express) + MySQL + Socket.io 提供数据与实时通讯服务。

## 2. 软件环境清单

在运行本项目之前，请确保您的开发或服务器环境已安装以下软件：

### 2.1 Node.js (必须)

本项目基于 Node.js 开发，必须安装 Node.js（推荐 v16.0.0 及以上版本）。

**下载地址：**

- Node.js 官方：https://nodejs.org/zh-cn/download/
- 或 NVM 安装：https://github.com/coreybutler/nvm-windows/releases

**安装验证：**

```bash
node -v
npm -v
```

**预期输出（类似）：**

```text
v16.18.0
8.19.2
```

**环境变量配置（Windows）：**

通常 Node.js 安装程序会自动配置环境变量。若未生效：
1. 新建系统变量 `NODE_PATH`，值为 Node.js 安装路径，例如：
   `C:\Program Files\nodejs\`
2. 编辑 `Path` 变量，添加：`%NODE_PATH%`

### 2.2 MySQL 数据库 (必须)

本项目使用 MySQL 作为数据存储，推荐安装 MySQL 8.0 或 MySQL 5.7。

**下载地址：** https://dev.mysql.com/downloads/installer/

### 2.3 Krpano 引擎环境 (按需)

本项目的前台全景漫游依赖 Krpano 引擎。项目内已打包了基础运行所需文件（`tour.js`, `tour.xml` 等核心插件），正常情况下**无需额外安装即可在浏览器中预览**。
若您需要对全景图进行深度二次开发（如使用 VTour Editor 或 Droplets 生成新场景），则需要下载并配置 Krpano 客户端。

**下载地址：** https://krpano.com/download/
**版本建议：** Krpano 1.20 或 1.21 版本。

### 2.4 Three.js (内置依赖)

项目中的 3D 模型展示使用了 Three.js。此为 Node.js 模块依赖，已配置在 `package.json` 中，执行 `npm install` 即可自动安装，无需额外全局配置环境。

## 3. 项目初始化与启动步骤

### 3.1 数据库导入配置

1. 打开MySQL管理工具（如 Navicat, MySQL Workbench 或命令行）。
2. 创建一个名为 `krpano` 的数据库，字符集推荐使用 `utf8mb4`。
3. 运行或导入本项目根目录下的 `krpano数据库.sql` 文件，完成表结构和初始测试数据的导入。
4. 进入 `server` 目录，打开 `.env` 文件，确认或修改数据库连接配置（如账号、密码）：
   ```env
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=root
   DB_NAME=krpano
   ```

### 3.2 启动后端服务 (Server)

后端服务负责提供 API 接口和 WebSocket 实时通讯服务。

1. 打开终端，进入 `server` 目录。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动服务：
   ```bash
   node index.js
   ```
   *服务默认运行在 `http://localhost:3000`*

### 3.3 启动前台全景展示端

前台主要面向访客，提供全景和 3D 模型展示。

1. 打开新的终端，进入项目根目录。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动服务：
   ```bash
   npm run dev
   ```
   *服务通常运行在 `http://localhost:5173`（具体端口请查看控制台输出）*

### 3.4 启动后台管理系统 (Admin)

后台管理端用于管理员或客服人员管理系统数据。

1. 打开新的终端，进入 `admin` 目录。
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动服务：
   ```bash
   npm run dev
   ```
   *服务通常运行在 `http://localhost:5174` 或 `5175`（具体端口请查看控制台输出）*

## 5. 项目核心功能说明

### 5.1 前台展示端功能

**360度全景漫游**

1. 访问前台首页，系统将自动加载全景场景。
2. 拖拽鼠标或滑动屏幕可 360 度无死角浏览企业展厅环境。
3. 点击场景内的热点（Hotspot）图标，可实现场景切换或触发特定交互事件。

**3D 模型交互预览**

4. 在全景场景中点击特定热点，可弹出 3D 模型预览窗口。
5. 支持对模型进行 360 度旋转、缩放查看产品细节。
6. 支持复杂机械模型的动画拆解展示，并附带动态数据参数面板（如转速、产能等信息）。

**实时客服沟通**

7. 点击右下角客服图标，展开实时通讯聊天框。
8. 访客无需登录即可发送消息，后台客服人员在线实时接收并回复，实现双向互动。

**用户留言板**

9. 点击界面左侧“留言”菜单，拉出留言墙面板。
10. 登录后可发表留言、查看其他访客的留言，并支持点赞互动。

### 5.2 后台管理端功能

**数据分析大屏**

11. 管理员登录后台后，首屏展示数据仪表盘。
12. 实时查看今日 PV/UV、平均停留时长，以及近7天/30天的访客数据趋势图表。

**客服聊天工作台**

13. 客服人员登录后进入“客服聊天”模块。
14. 左侧列表显示所有前台访客会话，带有未读消息红点提示。
15. 右侧窗口可与指定访客进行一对一的实时文字交流，并支持查看访客浏览历史。

**3D模型参数配置**

16. 进入“3D模型管理”模块，列表呈现前台展示的所有模型资产。
17. 支持动态编辑每个模型的参数信息（如批次、运行时长、产能等），修改后前台实时同步生效。
18. 支持开启或关闭特定模型的展示状态。

**系统与用户管理**

19. **留言墙管理**：审核前台用户留言，支持删除不当言论。
20. **用户管理**：管理员可添加、编辑系统账号，并分配“管理员”或“客服”角色权限。
21. **系统配置**：一键修改展厅的标题、Logo及外部跳转链接等全局基础配置。
